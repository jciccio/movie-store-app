# Video Store

## Technologies used

- Ruby On Rails
- ReactJS
- SQLite


## Basic Setup

### Back End

#### Pre Requisites
- Install Ruby 3.0.0 or 3.2.2

#### Configuration
- Run configuration on the Back End using `bundle install`
- Execute Migrations: `rake db:migrate`

#### Execution
- Run `rails s`

### Front End

#### Pre Requisites
- Install Node
- Install Yarn

#### Configuration
- Execute `yarn install`

#### Execution
- Execute `yarn start`
